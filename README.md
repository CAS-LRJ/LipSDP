# LipSDP

This repository contains code for the LipSDP framework introduced in
[LipSDP](https://arxiv.org/abs/1906.04893) by Mahyar Fazlyab, Alexander Robey,
Hamed Hassani, Manfred Morari, and George J. Pappas.
